# Eazi Cut

Marketing website for Eazi Cut — a luxury African menswear house.

## Stack

- Next.js 15 (App Router) · TypeScript (strict)
- Tailwind CSS · shadcn/ui primitives
- Framer Motion (editorial reveals)
- next/font (Playfair Display + Inter)

## Run

```bash
npm install
npm run dev
```

Visit http://localhost:3000

## Pages

- `/` Home
- `/collections` Collections
- `/lookbook` Lookbook
- `/about` Atelier
- `/contact` Contact

See `docs/` for product, brand, and design guidance.
