# Eazi Cut Brand Guidelines

**Version:** 1.0

**Project:** Eazi Cut Digital Platform

**Prepared By:** Mubeejoy Technologies

---

# Purpose

This document defines the visual identity, personality, and communication standards for Eazi Cut.

Every designer, frontend developer, AI assistant (Claude Code, Codex, ChatGPT, Cursor), photographer, and content writer must follow these guidelines.

Consistency builds trust.

Luxury brands never look different from one page to another.

---

# Brand Positioning

Eazi Cut is a premium African menswear and bespoke tailoring brand.

We do not compete on price.

We compete on:

• Craftsmanship

• Elegance

• Quality

• Experience

• Trust

• Personal Service

Every customer should feel they are buying into a premium lifestyle, not simply purchasing clothing.

# AI Design Rules

All AI tools must:

Read this document before generating UI.

Never invent new colors.

Never introduce new fonts.

Use the spacing system consistently.

Maintain the luxury aesthetic.

Prefer simplicity over decoration.

Every generated component must feel like it belongs to the same design system.

---

# Final Principle

Luxury is not created by adding more.

Luxury is achieved by removing everything unnecessary until only excellence remains.
