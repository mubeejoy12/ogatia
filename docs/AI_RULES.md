# AI Rules

Avoid:

- Generic AI aesthetics
- Purple gradients
- Startup landing pages
- Cookie-cutter dashboards
- Generic SaaS layouts

Prioritize:

- Luxury fashion aesthetics
- Editorial layouts
- Strong typography
- Large imagery
- Elegant whitespace

Every page should feel worthy of a premium menswear brand.

Ask before generating UI:

Would a customer trust this brand with a ₦500,000 wedding outfit?

If not, redesign it.
